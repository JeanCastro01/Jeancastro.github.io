Project information

  • Category: Shell Script
  • Project date: 19 July, 2023

Description

The create-cluster.sh script is designed to automate the creation of a Kubernetes cluster using Kind (Kubernetes in Docker). This tool simplifies the setup of a Kubernetes environment for development and testing purposes.

Features

  • Kubernetes Cluster Creation: Efficiently sets up a Kubernetes cluster with Kind.
  • Flexible Configuration: Allows you to include or exclude additional components such as Ingress controllers and MetalLB (a load-balancer implementation).
  • Customizable Cluster Naming: Provides the ability to specify a custom name for the cluster to differentiate it from other environments.

Usage

Run the script with various options to customize the cluster setup:

$ ./create-cluster.sh --no-ingress --no-metallb --cluster-name demo
  • --no-ingress: Excludes the Ingress controller setup from the cluster.
  • --no-metallb: Omits the MetalLB load-balancer setup.
  • --cluster-name demo: Sets the name of the Kubernetes cluster to "demo."

Tested On

Bash Version: 5.1.8

The script is tested to ensure compatibility with Bash version 5.1.8, but it should also work with other versions of Bash.

Example

Here is an example command to create a basic Kubernetes cluster with the default settings:

$ ./create-cluster.sh --cluster-name my-cluster

In this example, the cluster will be created with the default configurations for Ingress and MetalLB.

Additional Information

For more detailed usage and options, refer to the script documentation or run the script with the --help flag:

$ ./create-cluster.sh --help

This will display additional options and usage instructions.
